# VibeMaster: 2D MMORPG Adventure

A simple 2D top-down MMORPG where players can explore a world, complete quests, and interact with other players. This project blends Sci-fi, Fantasy, and Abstract themes with a "Polygons meets Zelda meets Graal Online meets Realm of the Mad God" aesthetic.
