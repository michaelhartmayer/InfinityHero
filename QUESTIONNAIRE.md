# Project Questionnaire: 2d MMORPG Adventure

Please fill out the following details to help guide the development of the project.

## 1. Core Gameplay
- **Movement Style**: Grid based, point and click
- **Combat System**: Real time TAB targeting skills can be used with 1 2 3 4 keys (different skills can be learned by different classes tbd)
- **Progression**: XP, RNG Items, Seperate JOB system for crafting progression
- **Interaction**: Chat, Trading, Parties

## 2. Visuals & Audio
- **Art Style**: Design this so we can easily upgrade the art style later. For now make everything out of simple but elegant polygons.
- **Perspective**: Top down
- **Theme**: Sci-fi Fantasy Abstract

## 3. Technical Preferences
- **Frontend Framework**: React + Vite + WebGL + TypeScript
- **Game Engine/Library**: Three.js
- **Backend/Multiplayer**: Node.js + Socket.io
- **Database**: Create a facade and implement in filesystem for now

## 4. Scope & MVP
- **What is the absolute minimum feature set for V1?**
    - [ ] World Maps

    - [ ] Inventory System
    - [ ] Experience, Health, and Energy System
    - [ ] Death + Respawn System
    - [ ] Leveling System + EXP Table
    - [ ] Loot Table System
    - [ ] Collision System
    - [ ] Chat
    - [ ] Job System COMING SOON
